# ExitLens — Exit Interview Sentiment Analysis Tool
An AI-powered HR analytics tool that automatically analyses exit interview responses, detects themes and sentiment, identifies patterns by department and manager, and generates actionable insights for HR leaders.

## Features
- Sentiment analysis using NLP
- Theme detection via keyword matching
- Department and manager-level breakdowns
- AI-generated risk flags
- Flight risk prediction for active employees
- Clean dashboard interface for HR leaders

## Tech Stack
- Python (pandas, VADER, scikit-learn) — data generation and NLP
- HTML, CSS, JavaScript — frontend dashboard
- JSON — pre-processed analysis output

## Version
Version 0 — Full Analysis tab complete. Predictions and Basic Analysis tabs in progress.

# Objective
HR Analytics Portfolio Project — targeting CHRO and VP People Analytics roles in Malaysia and SEA.
